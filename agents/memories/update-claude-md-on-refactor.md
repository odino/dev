## Update CLAUDE.md During Large Refactors

When performing a large refactoring in a project, always check and update `CLAUDE.md` (and any files it references) to ensure all documented paths, commands, patterns, and architectural descriptions remain accurate.

If anything referenced in `CLAUDE.md` is no longer valid after the refactor (e.g., moved files, renamed modules, changed commands, restructured directories), update it as part of the refactoring task — not as an afterthought.
